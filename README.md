## What is this project

This is a React project (CSR), built with TypeScript and React Testing Library.

## Demo
www.bowerclone.netlify.app

## Steps to follow:

#### Install dependencies
```sh
yarn install
```

#### Run the project locally
```sh
yarn start
```

#### Run the Tests
```sh
yarn test
```
